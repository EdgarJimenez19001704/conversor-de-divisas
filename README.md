# DivisApp – Conversor GTQ → USD

Aplicación con Ionic + Angular para convertir quetzales (GTQ) a dólares (USD) usando una tasa fija.

## Ejecutar (desarrollo)

```bash
npm install
ionic serve
```

Abre: [http://localhost:8100](http://localhost:8100)

## Compilar Android (resumen)

```bash
npm run build
npx cap copy android
npx cap open android
```

En Android Studio: Build → Generate Signed Bundle / APK… → elegir APK → release (crear keystore si es la primera vez).
